# Calcio Brand Assets

## Primary Logo

The primary Calcio app icon is `calcio-icon-primary.svg`: a deep midnight navy
rounded-square tile (`#071A3D`) with a warm white football/target mark
(`#F8F5ED`) and one champagne-gold winning-prediction segment (`#D4A44B`).

This navy target-ball icon is the source of truth for favicon, PWA icons,
install surfaces, and primary in-app logo usage.

## Palette

- Midnight navy: `#071A3D`
- Warm white: `#F8F5ED`
- Champagne gold: `#D4A44B`

## Variants

- `calcio-icon-primary.svg`: primary app icon.
- `calcio-icon-dark.svg`: same primary icon for splash/loading contexts.
- `calcio-icon-gold.svg`: secondary premium gold-tile variant.
- `calcio-mark.svg`: transparent mark with primary white/gold colours.
- `calcio-mark-mono.svg`: single-colour mark for simple UI usage.
- `calcio-mark-gold.svg`: transparent gold mark.

## Usage Rules

- Use `calcio-icon-primary.svg` as the master asset.
- Use the gold tile only as a secondary premium brand variant.
- Do not use AI-generated PNGs as the master asset.
- Do not add shadows, textures, outlines, gradients, or random recolours.
- Keep exports generated from the SVG source rather than hand-editing PNGs.
